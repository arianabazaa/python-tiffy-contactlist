# python-tiffy-contactlist

A contact management application developed in Python that allows users to add, modify, and delete contacts while storing data in a JSON file.

## Features
- Add new contacts
- Modify existing contacts
- Delete contacts
- Store contact data persistently using JSON
- Sort contacts alphabetically by last and first name
- Error handling for duplicate IDs and missing files

## Technologies Used
- Python
- JSON
- Object-Oriented Programming
- File I/O
- Dictionaries

## Skills Demonstrated
- CRUD operations
- Exception handling
- Data persistence
- Software design and organization
